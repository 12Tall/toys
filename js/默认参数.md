# 默认参数  

```js
// 这里以一个构造函数为例
function Obj({p1='p1',p2=2,...others}={}){
    // function(p1='p1'){}  // 表示参数的默认值
    // function({p1,p2}={})  // 解构赋值
    // function(...p)  // 多个参数  

    // 这里还没想到更简单的方法
    this.p1=p1;
    this.p2=p2;

    // 默认参数之外的参数获取
    for(var i in others){
        this[i] = others[i];
    }

    return this;
}
```
