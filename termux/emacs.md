# Emacs  

## 窗口布局  

    - 启动`emacs` 会进入默认欢迎界面  
    - 按`q` 进入编辑器模式  

## 键位  

### 普通键位  

`C-<>` 表示`Ctrl + <>`;`M-<>` 表示`Alt + <>`  

- `←/↑/↓/→` 和其他编辑器没有区别  
- `C- ↑↓←→/bfpn` 按字移动  
- `M- ←/→/ b/f` 会按词移动  
- `C- a/e` 行首/行尾；`M- a/e` 文首/文尾  
- `M-d` 向右删除；`C- k` 向右删除，直到行尾  
- `M- u/l/c` 切换大小写，首字母大写，按单词  
- `M-t` 交换两个单词；`C-t` 交换字母  
- [x] 可以总结`Alt` 对词操作，对应着有`Ctrl` 对字母操作

- `C-s` 保存  
- [x] `C-x C-c` 退出
